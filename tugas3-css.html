Subjudul dengan External CSS

Ini adalah contoh paragraf yang menggunakan external CSS. External CSS ditulis dalam file terpisah dengan ekstensi. CSS dan dihubungkan ke dokumen HTML menggunakan tag <link>. Metode ini paling efisien karena satu file CSS dapat digunakan oleh banyak halaman HTML, memudahkan maintenance dan konsistensi desain.
